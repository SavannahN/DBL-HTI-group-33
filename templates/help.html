The sample dataset, preloaded into the tool, contains subway maps from different cities and data collected from several different users. You can use the tool by choosing a map, user, and method(s). Alternatively, you can upload your own data, which will change the map and user options.

Maps

These are the stimuli you can choose from.

  • 01 vs 01b: the location of the target on the map.
  • S1: color, S2: grayscale

We are still working on using the slider to display a grayscale map.

User

These are the users from the dataset that data was collected from.

Visualization

You can choose which method to display. Note that AOI Rivers only works for all users while Data Table only works for a single user.

Number of AOIs

This determines how many AOIs are used, which applies to the transition graph and the AOI rivers.

Visualisation methods

Data table: The data table method produces a table that can be sorted, ascending and descending, by each variable.

Gazeplot: The gazeplot method shows the chosen stimulus with the saccades, as lines, and fixations, as circles, plotted onto it. The size of the circles represent the relative fixation duration.

Static Visual Attention Map (Heat Map): The static visual attention map is a map that displays the relative fixation duration putting a color filter over the stimulus. Blue areas are not fixated on at all and red areas are fixated on the most.

Gazeplot + Heat Map: This map shows the stimulus with the gaze plot and heatmap plotted over it.

Transition Graph: The transition graph method displays a graph in which each node represents an area of interest and each edge represents the relation between to AOIs. The size of the arrow represents how much participants shift from the one AOI to the other.

Gaze Stripes: The gaze stripes method shows each of the fixations on the stimulus of a user ordered as a one long stripe.

AOI Rivers: The AOI rivers give a plot in which the AOIs are all represented as a colored stripe and the thickness of the stripe in a particular moment in time is the participants's relative interest in that area at that time.
