The sample dataset, preloaded into the tool, contains subway maps from different cities and data collected from several different users. You can use the tool by choosing a map, user, and method(s). Alternatively, you can upload your own data, which will change the map and user options.

  • Identifies two unique tasks (S1 and S2) with different starting points and targets on the map.

Visualization

The available methods with which the eye movement data can be visualized.

    Notes:
  • - The Data Table only works for a single user.
  • - The AOI Rivers only works for all users.
Number of AOIs

The number of Areas Of Interest (AOIs) are applied to the Transition Graph, AOI map and AOI rivers.

    Notes:
  • - The Transition Graph may show fewer AOIs by removed nodes from the visualization weak or no edges attached to it.
Visualization methods

Data table: The data table method produces a table that can be sorted, ascending and descending, by each variable. The data table cannot be displayed in combination with other visualizations.

Gaze Plot: The gazeplot method plots the saccades as lines and fixations as circles on the stimulus. The size of the circles represent the relative fixation duration.

Static Visual Attention Map (Heat Map): The static visual attention map is a heat map that indicates the relative fixation duration with colors on the stimulus. The colors range from red, areas that are fixated on the most to blue, areas that fixated on the least.

Static Visual Attention Map + Heat Map: A visualization combining the Static Visual Attenion Map and the Gaze Plot in one graph.

Transition Graph: The transition graph method is a weighted directed graph in which each node represents an area of interest and each edge represents the strength of transitions between the AOIs. The size of each edge represents how much participants relatively shift from one AOI to the other.

Gaze Stripes: The Gaze Stripes displays strips of images that show fixations of users in sequential order.

AOI Map: The AOI Map displays AOIs on the stimulus map by means of the k-nearest neighbour algorithm. It is particularly useful to display in combination with the Transition Graph and AOI Rivers.

AOI Rivers: The AOI rivers illustrate a plot in which the AOIs are represented as a so-called river, where its thickness at a particular moment in time is the total number of unique users fixated on that area over time. The visualization can only be displayed for all users, where time is normalized.
