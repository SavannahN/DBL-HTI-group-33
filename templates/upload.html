Formatting guidelines

Both the .zip and the .csv file need to be structured specifically. following the upcoming guidelines.
There is no way to edit the files after uploading. Make sure they are in the right formating beforehand.
The .csv file needs to have a set of columns with the first row being titles.
Columns from left to right.
*Timestamp: Time stamp in second can be any positive integer
*StimuliName: Full file path & FileName for the image that it corresponds to (including extension)
*FixationIndex: can be any string
*FixationDuration: Time length in second can be any positive integer
*MappedFixationPointX: x coordinates can be any integer (left is x 0)
*MappedFixationPointY: y coordinates can be any integer (top is y 0)
*user: Name of the user can be any string
*description: Personal comments NOT used by our code. may be empty


The .zip file needs to contain an image for each StimuliName used.
If all images are kept in the top folder you do not need to add a file path to the StimuliName column in .csv
.jpg and .png have been tested other file formats may work but are not supported
